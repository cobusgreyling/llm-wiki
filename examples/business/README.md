# Business wiki example

A minimal team knowledge base built from meeting transcripts, a customer call, and an internal roadmap doc. Use it as a template for business/team wikis.

```bash
wiki --root examples/business search "enterprise"
wiki --root examples/business ingest-status
wiki --root examples/business expand synthesis --section "Thesis"
```

Open `examples/business/` as an Obsidian vault alongside the other examples.
